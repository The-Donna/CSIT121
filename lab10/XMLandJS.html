Learning about XML and JS.

What is XML?

XML is a programming language that is short for eXtensible Markup Language. It was created to store and transport data. It supports information exchange between computer systems.It has no predefined tags therefore one can create their own tags hence, it is self-descriptive.

It was designed to separate information from presentation meaning that it focuses on the information that is on the webpage rather than how it looks. It was designed to be both human and machine readable.

How is XML used?

XML can be used for various things which are listed below:

  1. Data Transfer: It aids the easy transfer of data between different applications, browsers etc.
  2. E-commerce: It is used in the E-commerce industry to represent product catalogs and invoices among others in a simplified format that aids the easy exchange between different systems.
  3. General applications: It provides a standard method to access information thereby making it easier for all kinds of applications, platforms and devices to use, store, transmit andd display data.
  4. Layout Creation: XML helps determine where certain data should be on a screen

These are just a few uses of XML. it is widely recognized for it's easy storage and data transfer.

What is JavaScript?

JavaScript is an object-based, client-side scripting language. This is a language that runs on your browser rather than from a server. This is what actually makes it peculiar. It shouldn't be confused with Java since they are both quite different. They apparently have more differnces than similarities.

It provides interactivity, real-time content updates and also provides animation. It easily provides these functions since it doesn't have to reload the whole page to provide them. It is embedded in a HTML page with a script tag

How is JavaScript related to HTML and CSS?

JavaScript is related to HTML and CSS in the sense that they work hand in hand to make webpages that are structured, appealing and interactive. HTML provides the structure and content for webpages, CSS adds style and visual designs to the webpage while JavaScript builds upon the contents to make them interactive and engaging. These three languages work together to create many of the websites we visit today and are the the three essential components of modern web development.

What are some common JavaScript Elements, Properties, Methods, and Functions?

Some common Elements, Properties, Methods, and Functions of JavaScript include:

  1. Elements:
    • Document Object Model: It is constructed as a tree of objects. It allows JavaScript to make changes to the content and style of the page. It gives JavaScript the power it needs to create a dynamic HTML page.
    • Event: It represents an action that is made by the user that can be detected and responded to by the JavaScript. The events happen to HTMl elements and JavaScript reacts to these events.
  2. Properties:
    • Length: This property returns the lenght of a string. The lenght property of an empty string is 0.
    • ClassName: It returns the CSS class of an element.
    • ClassList: This returns the CSS ClassNames of an element.
  3. Methods:
    • Filter: This method is used to create a new arrray that contains all the elements from the original array that meet certain conditions.
    • SetInterval: It is used to repeatedly run a function at certain intervals.
  4. Functions:
    • Eval: It is used to evaluate or execute a string as if it was a JavaScript code.
    • ParseInt: It is used to convert a string to a floating-point number.
